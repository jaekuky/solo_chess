# 체스 사운드 파일

이 폴더에 아래 MP3 파일을 배치하면 게임에서 효과음이 재생됩니다.

## 필요한 파일 목록

| 파일명 | 용도 |
|--------|------|
| move.mp3 | 일반 기물 이동 | 
| capture.mp3 | 기물 잡기 | 
| check.mp3 | 체크 | 
| castle.mp3 | 캐슬링 | 
| promote.mp3 | 프로모션(승격) | 
| game-start.mp3 | 게임 시작 | 
| game-end.mp3 | 게임 종료 |
| victory.mp3 | 승리 | 
| defeat.mp3 | 패배 | 
| draw.mp3 | 무승부 | 
| illegal.mp3 | 잘못된 수 | 
| timer-warning.mp3 | 타이머 경고 | 
| hint.mp3 | 힌트 | 

## 참고

- 무료 체스 사운드는 [Lichess](https://github.com/ornicar/lila/tree/master/public/sound) 등에서 참고할 수 있습니다.
- 파일이 없어도 앱은 정상 동작하며, 재생 실패 시 콘솔에 경고만 출력됩니다.
